• Lightweight file manager built with pure PHP, Bootstrap 5, and MySQL
• Supports uploading files
• Supports previewing files directly in the browser
• Supports downloading files
• Supports editing file metadata
• Supports deleting files
• Simple and clean user interface
• Minimal codebase designed for easy extension
• Includes Bootstrap UI
• Shows image thumbnails
• Supports inline preview for images, PDFs, text files, audio, and video
• Includes pagination
• All user-facing pages are stored in the public folder
• Uploaded files are stored in the uploads folder
• Database connection is stored in the src folder
• .gitkeep keeps the uploads folder tracked in Git
• Database requires a filemanager database and a files table
• Project runs using PHP’s built-in server
• Preview supports jpg, png, gif, webp, pdf, txt, csv, log, json, mp3, wav, mp4, webm
• Unsupported formats (zip, rar, docx, xlsx, exe) download automatically
• Can be extended with file type icons
• Can be extended with drag-and-drop upload
• Can be extended with user authentication
• Can be extended with folder support
• Can be extended with Bootstrap modal previews
• Can be extended with syntax highlighting for code files
• Licensed under MIT
• Free to use, modify, and learn from
